# Health API
## _Based on Python and FastAPI

Library used:
- fastapi
- python-jose
- passlib
- autopep8
- fastapi-pagination

## Features

- Login, get all users, get user, update user...
- Meal types, meal history...


## Envinroment

```sh
cd /health-api
python3 -m venv ./env
```


## Installation

```sh
pip3 install -r requirements.txt
```

```sh
cd /app
uvicorn main:app --reload
```

```sh
127.0.0.1:8000
```
